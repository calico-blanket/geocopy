# GeoCopy 開発進捗管理 (PROGRESS.md)

## 📋 プロジェクト概要
地図から緯度・経度を素早く取得・コピーするための軽量ユーティリティアプリ。PC拡張機能、スマホPWA、Webの3形態で利用可能。

## ✅ 完了したタスク

### Phase 1: 基本機能の実装
- [x] Leaflet.jsを用いたインタラクティブ地図の表示
- [x] OpenStreetMap Nominatim APIによる場所検索機能
- [x] 地図タップおよびピンのドラッグによる座標取得
- [x] 緯度と経度の個別コピー機能（クリップボードAPI）
- [x] グラスモーフィズムを採用したモダンなUIデザイン

### Phase 2: プラットフォーム対応
- [x] **PWA (Progressive Web App)**: サービスワーカーとマニフェストの実装
- [x] **Chrome拡張機能**: ポップアップ形式（Vercelラッパー版）での実装
- [x] **Vercelデプロイ**: `geocopy.vercel.app` への自動公開設定
- [x] **GitHub連携**: `calico-blanket/geocopy` リポジトリでのバージョン管理

### Phase 3: ブラッシュアップと法的整備
- [x] **地図タイルの日本語化**: OSM Japanタイルへの切り替え（簡体字表記の解消）
- [x] **著作権表示の追加**: 地図データおよびタイル提供元のクレジット表示
- [x] **ドキュメント整備**: README.md の作成
- [x] **ライセンス設定**: MIT License (著作権者: calico_blanket) の適用

## 🚀 現在のステータス
- **Web版**: 公開中 ([geocopy.vercel.app](https://geocopy.vercel.app/))
- **拡張機能**: ローカルディレクトリから読み込み済み
- **モバイル**: PWAとしてホーム画面追加可能
