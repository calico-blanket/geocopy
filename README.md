# GeoCopy 📍

地図上の任意の地点の**緯度**と**経度**を簡単に抽出し、個別にクリップボードへコピーできるツールです。

## 機能

- 🗺️ **マップクリック**: 地図上をタップ/クリックして座標を取得
- 🔍 **場所検索**: 地名や住所で検索して地点へ移動
- 📋 **個別コピー**: 緯度・経度をそれぞれ独立してコピー
- 📱 **スマホ対応**: PWAとしてホーム画面に追加可能
- 🖥️ **Chrome拡張機能**: PCではブラウザの拡張機能としても利用可能

## 使い方

### Webアプリとして（スマホ・PC）

Vercelでホスティングされた URL にアクセスするだけで使えます。

### Chrome拡張機能として（PC）

1. このリポジトリをダウンロード（またはクローン）します
2. Chrome の `chrome://extensions/` を開きます
3. 右上の「デベロッパーモード」をオンにします
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択します

## 技術スタック

- HTML5 / CSS3 / JavaScript (ES6+)
- [Leaflet.js](https://leafletjs.com/) - 地図ライブラリ
- [CartoDB Voyager](https://carto.com/basemaps/) - マップタイル
- [OpenStreetMap Nominatim](https://nominatim.org/) - 住所検索API
